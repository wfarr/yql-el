YQL.el

About

YQL.el is a simple interface to YQL, a developer tool with a super-neat API which provides access to many Yahoo! web services (such as flickr). The goal is to provide a simple, somewhat Lispy way to interact with YQL without having to load up a browser (a lot of Emacs users live there -- trust us!).

This project was implemented as part of Yahoo! University Hack Day at Georgia Tech, March 2009. We were reimbursed with lots of free caffeine and sugar. We would like to thank the Yahoo! guys for rocking (and for the drinks and food). Our dentists are probably very upset with them though.
